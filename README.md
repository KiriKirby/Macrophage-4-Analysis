# Macrophage Image Four-Factor Analysis

Language: [中文](README.zh-CN.md) | [English](README.en.md) | [日本語](README.ja.md)

A Fiji/ImageJ macro for semi-automated quantitative analysis of macrophage images with beads (v2.2).

Key features:
- Manual cell ROI annotation with robust ROI handling
- Dual-path bead detection (threshold + edge) with strictness control
- Target bead sampling, optional exclusion filtering, and clump estimation
- Four-factor statistics with optional tiny-uptake adjustment
- Data formatting + optimization (PN/F parsing, IBR/PCR, eIBR/ePCR)
